## Research #1 - algorithms_benchmarks

1. Статистическая проверка гипотез (fixed-sized эксперимент)

### Цель исследования:

1. Ошибка первого рода
2. Ошибка второго рода
3. Требуемый объем наблюдений

### Решение:
